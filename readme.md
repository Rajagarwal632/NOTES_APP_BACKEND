Notes App Backend

A secure backend API for managing personal notes with authentication, archiving, and search functionality. This project is built as a learner-friendly but production-style backend using Node.js, Express, MongoDB, and JWT.
Features
User Signup & Signin (JWT based authentication)
Create private notes (user-specific)
Get all notes of a logged-in user
Update notes
Delete notes
Archive / Unarchive notes
Search notes by keyword (title + content)
Tech Stack
Node.js
Express.js
MongoDB + Mongoose
JWT Authentication
Zod (request validation)
dotenv (environment variables)
